# Project: Subway-Reservation
* 임산부, 장애인 등 사회적 약자를 위한 지하철 좌석 예약 애플리케이션
* 제 18회 임베디드 소프트웨어 경진대회
* 주최 : 
* 주관 : 
* 2020.06 ~ 2020.09

## 0. Overview
사회적 약자 및 임산부의 불편함을 해소하는 데에 기능적 목표를 두고 있습니다.
나아가 해당 애플리케이션 SRA를 통해 지하철을 타는 일반 승객들도 더 이상 양보가 의무라는 강박을 떨쳐버릴 수 있고,
양보를 받는 승객들 또한 눈치를 보지 않고 당당하게 탈 수 있는 대중교통 문화를 만드는 것이 최종 목표입니다.
<img src="/doc/imgs/overview.png" width="100%">

## 1. Goals
- (대구)지하철을 타 보면, 대부분의 사람들이 분홍색으로 칠해진 임산부 지정석에 앉기 망설여 질 것입니다. 또한 아직 배가 부르지 않은 임산부의 입장에서, 만석인 지하철 안에서 자신을 임산부라 밝히고 자리를 양보해 달라고 양보를 요청을 하기도 상당히 힘든 일입니다.
- 모든 지하철이 임산부, 노약자 배려석을 보유하고는 있지만 사용하는 사람 대비 낭비되는 자석이 많아 좌석 효율이 떨어집니다. 그리고 SNS나 여러 매스컴을 통해서 보도되는 지하철 내에서의 승객과의 갈등도 심심찮게 관찰할 수 있습니다. 따라서 현 지하철 내에 임산부 석과 노약자석에 대한 변화가 필요할 것입니다.

## 2. Function
- 로그인/회원가입 기능
- 좌석 예약 기능
- 중복 예약 좌석 확인 알고리즘
- LED 출력 기능
